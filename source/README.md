## デザインルールの公式ブログです。


### 環境構築
```bash
git clone 
npm install
```

### 記事を作成する

```
npm run create "記事タイトル"
```

### サーバを起動

```
npm run start 
```


### 記事の公開

```
npm run deploy
```

### 静的ファイルの生成

```
npm run generate
```


### npm run と　ローカルへのhexoインストールについて（メモ）
- 私の環境では、ローカルにhexoをインストールしているため、
```aidl
./node_modules/.bin/hexo  
```
と実行する必要がある。
